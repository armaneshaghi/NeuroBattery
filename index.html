NeuroBattery

Example of how to use ANTs to process multiple MR image types for a single subject. Here we provide input data for a single subject for which 5 different images were acquired in a single session: MPRAGE, pseudo continuous arterial spin labeling (PCASL), diffusion weighted/tensor, magnetization transfer, and resting state BOLD. Additionally we provide a template and associated labels/priors that we use as an anatomical basis for processing.

The general concept is to use the anatomical information in the T1 images as a spatial frame of refenece. Accoringly we find a spatial mapping between each subject's T1 image and the T1 template. Later, for additional MR images, we will perform intrasubject alignment to the T1 allowing for mapping to template space by composing multiple transforms and retaining the concept of relying on the T1 images for anatomical refenence. After aligning the T1 image to the template we generate various images relating to anatomical properties of the brain.

T1 (MPRAGE) derived data

  1. Brain extraction mask
  2. 4-tissue segmentation (CSF,white matter, cortex, and deep gray matter)
  3. Cortical thickness

In the script directory, process_t1.sh provides an example call to antsCorticalThickness.sh which is used to achieve the steps listed above. The full command is:

sh antsCorticalThickness.sh -d 3 \
-a ../data/input/PEDS012/20121031/Anatomy/PEDS012_20121031_mprage_t1.nii.gz \
-e ../data/template/T_template.nii.gz \
-m ../data/template/T_templateProbabilityMask.nii.gz \
-p ../data/template/Priors/priors%d.nii.gz \
-t ../data/template/T_template.nii.gz \
-k 1 \
-n 3 \
-w 0.25 \
-o ../data/output/PEDS012/20121031/PEDS012_20121031_
