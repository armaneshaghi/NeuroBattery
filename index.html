NeuroBattery

Here we provide an working example that illustrates the use of Advanced Normalization Tools (ANTs) to process multiple MR image types for a single subject. We provide and example input data for a single subject for which 5 different images were acquired in a single scanning session: T1 (MPRAGE), pseudo continuous arterial spin labeling (PCASL), diffusion weighted/tensor, magnetization transfer, and resting state BOLD. Additionally we provide a template and associated labels & priors that we use as an anatomical basis for processing.

The general concept is to use the anatomical information in the T1 images as a spatial frame-of-refenece. Accoringly we find a spatial mapping between each subject's T1 image and the T1 template. Later, for additional MR images, we will perform intrasubject alignment to the T1 allowing for mapping to template space by composing multiple transforms and retaining the concept of relying on the T1 images for anatomical referenence for both intra and inter subject exploration of the data. After aligning the T1 image to the template we generate various images relating to anatomical properties of the brain.

T1 (MPRAGE) image processing

  1. Brain extraction mask
  2. 4-tissue segmentation (CSF,white matter, cortex, and deep gray matter)
  3. Cortical thickness

In the script directory, process_t1.sh provides an example call to antsCorticalThickness.sh which is used to achieve the steps listed above. The full command is:

sh antsCorticalThickness.sh -d 3 \
-a ../data/input/PEDS012/20121031/Anatomy/PEDS012_20121031_mprage_t1.nii.gz \
-e ../data/template/T_template.nii.gz \
-m ../data/template/T_templateProbabilityMask.nii.gz \
-p ../data/template/Priors/priors%d.nii.gz \
-t ../data/template/T_template.nii.gz \
-k 1 \
-n 3 \
-w 0.25 \
-o ../data/output/PEDS012/20121031/PEDS012_20121031_

Having processed the T1 image, we want to continue to use this data as our anatomical frame-of-reference. The processing of the additional data will all follow a similar pattern: 1. Initial preprocessing to obtain a derived scalar image with relevant anatomical detail 2. Alignment of the derived scalar to the T1 data 3. Propogation of relevant labels, such as the brain mask, etc. 4. Quantification of modality specific metrics of interest.
